## How to run the application

### On Mac/Linux
##### $ cd myproject
##### $ python3 -m venv .venv
##### $ . .venv/bin/activate
##### $ flask run

### On Windows
##### > cd myproject
##### > py -3 -m venv .venv
##### > .venv\Scripts\activate
##### > flask run


### The app will be running on http://127.0.0.1:5000
